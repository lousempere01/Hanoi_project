Tour de Hanoï

Coups : 0
Clique une tour source puis une tour destination.
  • Le jeu comporte trois tours.
  • Les disques commencent tous sur la première tour.
  • Un seul disque peut être déplacé à la fois.
  • Un disque ne peut jamais être posé sur un disque plus petit.
  • L’objectif est de déplacer tous les disques sur la troisième tour.
